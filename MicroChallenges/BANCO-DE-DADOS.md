```markdown
# Banco de Dados - Iniciante

## Descrição

Introdução ao uso de banco de dados para organização e entendimento dos gastos do dia a dia, utilizando SQL e um SGBD.

---

## CBL

### Mentor Driven Challenge

Organização de dados financeiros utilizando banco de dados e consultas.

### Big Idea

**Saúde Financeira**

### Essential Question

> Precisa ser um especialista para controlar os gastos financeiros?

### Challenge

Melhorar a organização financeira para ter maior controle de gastos.

**Como criar uma solução que permita registrar, organizar e analisar gastos pessoais de forma eficiente ao longo do tempo?**

---

## Duração

**2 semanas**

---

## Grupos

- 2 a 3 estudantes por grupo.

---

## Entregáveis

- Board com Guiding Questions.
- Documento organizado com as respostas e fontes utilizadas para responder às Guiding Questions.
- Uma ou mais tabelas criadas e organizadas em um SGBD.
- Documento com insights sobre os dados.
- Consultas realizadas para responder às perguntas.
- Identificar pelo menos um caso de erro.

---

## Metodologia

- Atividades práticas em grupo.
- Mentoria para apoiar dúvidas técnicas.
- Checkpoints com os grupos para validar o entendimento dos dados, consultas e relacionamentos.
- Discussão dos resultados e apresentação.

---

## Constraints (Obrigatoriedades)

- Utilizar um SGBD (**Postgres**).
- Utilizar **SQL**.
- Utilizar comandos como:
  - `SELECT`
  - `WHERE`
  - e comandos relacionados.

---

## Cronograma

### Semana 1

#### Aula 01

**Engage** do challenge e **Investigate**:

- O que são dados e como são armazenados.
- Discussão sobre controle de gastos.

#### Aula 02

**Checkpoint 1** (horário marcado com os grupos):

- **Synthesis** do Investigate.
- Workshop, se necessário.

---

### Semana 2

#### Aula 03

- Mentoria (mentor circulando entre os grupos).
- **Action**:
  - Criação da tabela.
  - Inserção de dados.
  - Primeiras consultas.

#### Aula 04

- Mentoria (mentor circulando entre os grupos).
- Entrega da atividade.
```
